About Scriptbill Withdrawal

Scriptbill Uses a Decentralized Withdrawal and Deposit System, instead of an exchange system to determine the value of her credit. This means when you withdraw a Scriptbill Credit, a credit trasaction block is inserted into the Scriptbill database which is decentralized for a matching depositor. The depositor finds the withdrawal request and pays directly into the account you set for the withdrawal.

Scriptbill uses agreement to secure this withdrawal transactional request in the network. This means even after the depositor has made the deposit into the account, the withdrawer must confirm this deposit before the depositor will inherit the credit.

Scriptbank helps Scriptbill users have a more smoother withdrawal experience by helping Scriptbill users fulfill their withdrawer request on a little transaction processing fees. This fees is neccessary to help in processing the transaction for you to make your withdrawer request quicker and smoother.

About Scriptbill Deposits

Scriptbill Uses a Decentralized Withdrawal and Deposit System, instead of an exchange system to determine the value of her credit. This means when you demand for a Scriptbill Credit, a credit trasaction block is inserted into the Scriptbill database which is decentralized after finding a matching withdrawer. The depositor finds the withdrawal request and pays directly into the account the withdrawer set for the withdrawal.

The challenge is there is no way Scriptbill can verify deposits made using credits not registered in the Scriptbill network. This makes it impossible to create agreement for your deposit transaction. However, the credit withdrawn can't be recovered back by the withdrawer when there is a complimenting deposit transaction in the Scriptbill decentralized database system. The withdrawer who created the agreement in this transaction must create a cancellation request to the depositor, which must be signed or appealled by the depositor to get back his withdrawn credit if the deposit could not be confirmed.
